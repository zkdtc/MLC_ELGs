General Description
This file gives the classification of 0.32<z<0.8 emission line galaxies with good [OIII], [OII], Hb stellar velocity dispersion, u, g, r, i, z magnitudes measurements using the random forest classifier developed in Zhang et al. (2019). Project site is https://github.com/zkdtc/MLC_ELGs. Code V1_0 is used For DR16. Contact zkdtckk@gmail.com for more informations and most up-to-date version.
